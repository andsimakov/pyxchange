Вам необходимо реализовать сервис для обмена картинками.

Страницы
/  главная - форма для загрузки и галерея последних загруженных картинок
/popular - галерея популярных картинок
/<key> - страница отдельной картинки

Требования к модели для хранения картинки

Поля
- картинка (ImageField)
- описание (текстовое поле, опционально)
- ключ (SlugField, должен быть уникальным)
- дата/время загрузки
- дата/время последнего просмотра
- счетчик просмотров

Методы, которые необходимо реализовать
- get_absolute_url

Логика работы:

Для  каждой загруженной картинки генерируется уникальный ключ (можете воспользоваться тем же алгоритмом, который использовался для сокращенных URL). После успешной загрузки происходит редирект на страницу картинки. На странице картинки должна отображаться следующая информация:
- картинка (тег img)
- описание
- дата загрузки
- размер файла
- размер (ширина, высота)
- к-во просмотров
При каждом заходе на страницу картинки счетчик просмотров увеличивается на 1.

На главной странице отображаются 12 последних загруженных картинок, отсортированных по времени загрузки в обратном порядке (последние сверху). 

На странице /popular отображаются 12 картинок с наибольшим количеством просмотров, отсортированные по к-ву просмотров в обратном порядке

Все картинки на главной странице и списке популярных должны быть кликабельным ссылками, ведущими на страницу картинки.

Для работы с картинками вам понадобится библиотека Pillow. Для  более сложных манипуляций, например создание thumbnails, можно воспользоваться библиотекой django-imagekit.


Проект должен быть оформлен как git репозиторий и содержать файл requirements.txt со списком библиотек.



Бонусные задачи:

1. Реализовать аутентификацию с помощью django.contrib.auth, опциональную привязку картинок к пользователю, и страницу кабинета для аутентифицированного пользователя со списком всех загруженных им картинок и возможностью их удаления.

2. Реализовать простую систему "лайков":
 - добавить соответствующее поле к модели картинки
 - форму (или скрипт) для отправки лайка со страницы картинки
 - страницу самых понравившихся картинок.
